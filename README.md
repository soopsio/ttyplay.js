# TTYPlay.js

`ttyplay` in JavaScript.

## Example

``` js
new TTYPlay('some-id', {
  url: '/data/ttyrecord.gz',
  gzip: true,
  cols: 80,
  rows: 25
});
```

For a more detailed example, check:

- [TTYPlay.org](http://ttyplay.org/)
- [The documentation](http://ttyplay.org/ttyplay.js/)
